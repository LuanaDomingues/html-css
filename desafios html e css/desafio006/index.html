Tags em HTML5

Em HTML5 podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso:

  • Parágrafo

    • Podemos criar vários paragráfos.

  • Títulos

    • Existe uma hierarquia dentro das tags de títulos, podemos usar até 6 níveis de títulos.

  • Imagens

    • Podemos adicionar várias imagens no nosso site, o melhor tipo de imagem para colocarmos em nosso site é o formato JPEG e o formato PNG.

  • Abreviações

    • Serve para quando queremos escrever uma sigla e deixar o significado ao usuário.

  • Código fonte

    • Pode delimitar o seu código. A principal vantagem dessa tag é o valor semântico que ela representa, indicando ao navegador que se trata de um código de computador.

  • Listas numeradas

    • Também chamada de listas ordenadas, são aquelas listas em que as ordens dos itens são importantes.

  • Listas com demarcadores

    • Também chamadas de listas com marcadores, são listas onde a ordem dos itens não influencia no significado da lista.
