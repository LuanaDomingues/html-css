Desafio das mensagens

Podemos usar quebra de linhas de duas formas, com a tag <BR> e com a tag <PRE>.

Qual a diferença entre a tag <BR> e a tag <PRE>?

  • Tag <BR>

    • A tag <BR> é usada para quebrar linhas, independente do lugar em que ela esteja, ela vai quebrar uma linha mesmo que ela esteja dentro de um parágrafo. Exemplo:

    *Um
    ** Dois
    ***Três
    ****Quatro

    ****Quatro
    ***Três
    **Dois
    *Um

  • Tag <PRE>

    • A tag <PRE> é uma tag pré-formatada, da forma que escrevermos o nosso código com essa tag, dessa forma aparecerá em nosso site. Exemplo:

        *Um
     **Dois
      ***Três
       ****Quatro
       
       ****Quatro
      ***Três
      **Dois
     *Um
