Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <B> (Não semântica).

Nesta frase, temos um termo em destaque usando a tag <STRONG> (Semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (Não semântica).

Nesta frase, temos um termo em ênfase usando a tag <EM> (Semântica).

Texto marcado

Para criar um texto marcado usamos a tag <MARK>.

Para mudar a cor da marcação podemos usar STYLE do CSS dentro da marcação <MARK>.

Podemos também criar um seletor dentro do <HEAD> para mudar todas as marcações existentes na página.

Texto grande e pequeno

Antigamente se usava a tag <BIG> para deixar um texto maior, mas hoje ela está OBSOLETA.

A tag <SMALL> é usada para deixar o texto menor, e ela ainda é usada no HTML5.

Texto deletado

Antigamente se usava a tag <STRIKE> para criar um texto deletado (Não semântica).

Atualmente usamos a tag <DEL> que significa que o texto pode até mesmo ser lido, mas deve ser desconsiderado.

Texto inserido

A tag <U> era usada antigamente para sublinhar um texto (Não semântica).

A tag <INS> significa que o texto sublinhado deve ter uma atenção, deve ser lido.

Texto sobrescrito

Para criar um número pequeno sobre um texto devemos usar a tag <SUP>, exemplo: 20-x2

Texto subscrito

Para criar um número pequeno abaixo do texto usamos a tag <SUB>, exemplo: H2O
