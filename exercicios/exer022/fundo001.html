Personalizando fundos

Vimos no primeiro capítulo deste curso que uma das maneiras de personalizar o fundo do nosso site é usando o <background-color>.

A segunda maneira para personalizarmos fundos é fazendo um degradê, usando a tag <background-image> com a função linear-gradient.

Agora neste terceiro capítulo aprenderemos a usar imagens para fundos, que de certa forma é uma maneira NÃO semântica, é uma forma de estilizar o nosso site usando a mesma propriedade para fazer degradê <background-image> muda apenas a função, que é url.
